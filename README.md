FFmpeg_Eagle
============

The source code and build scripts of FFmpeg used in Eagle

Please define the following environment variables for the build script build_ffmpeg_android.sh:
* ANDROID_NDK_ROOT
* ANDROID_TOOLCHAIN
* ANDROID_SYSROOT
* PLATFORM
* COMPILER_NUM_THREADS
* LIBRARY_UPDATE
